Onchain SVG Minter SVGM

A portable agent skill for generating, validating, and minting fully-onchain SVG NFTs on the Pharos Network — no IPFS, no external hosting, no broken art links. Everything lives in the contract's tokenURI and renders as a single base64 data: URI.

What it does

Three steps, all onchain, all reproducible by an AI agent.

① Generate

Deterministic SVG generator. Same seed → same artwork. The agent passes shapes, palette, and a seed; the skill returns a well-formed SVG string.

② Validate

Strips <script> tags, javascript: URIs, and on*= event handlers. Enforces a 24 KB cap so every mint fits comfortably inside a Pharos block.

③ Mint

Calls OnchainSVG.mint(to, svg) via viem. The contract stores the SVG, builds a base64 JSON, and returns it from tokenURI. The agent never touches a host.

④ Read

read-token.ts decodes the onchain tokenURI into a JSON metadata object and a ready-to-render data:image/svg+xml;base64,… image.

Agent Workflow

A real agent runs this exact 4-step sequence when the user says "mint an onchain SVG NFT on Pharos." Click any step to expand the simulated viem call.

1
Generate SVG
shapes=[circle,rect] · palette=[#2F80ED,#0D0D0D] · seed=42
// agent/scripts/generate-svg.ts
import { generateSVG } from "./agent/scripts/generate-svg";

const svg = await generateSVG({
  shapes:  ["circle", "rect", "polygon"],
  palette: ["#2F80ED", "#0D0D0D", "#F5F5F5"],
  seed:    42,
  size:    512,
});
// → "<?xml version="1.0"...<svg xmlns="http://www.w3.org/2000/svg"...>"
2
Validate (size + safety)
2.1 KB · no <script> · no on*= · no javascript:
import { validateSVG } from "./agent/scripts/validate-svg";

const v = validateSVG(svg);
// → { ok: true, errors: [], sizeBytes: 2153 }

if (!v.ok) throw new Error(v.errors.join("; "));
3
Send mint transaction on Pharos Mainnet
viem · chain 1672 · gas ~218,403
import { createPublicClient, createWalletClient, http, parseAbi, encodeFunctionData }
  from "viem";
import { pharosMainnet } from "./agent/lib/pharos";

const abi = parseAbi(["function mint(address to, string svg) external returns (uint256)"]);

const data = encodeFunctionData({
  abi, functionName: "mint",
  args: ["0xRecipient", svg],
});

const txHash = await walletClient.sendTransaction({
  account, to: "0xYourCollection", data, chain: pharosMainnet,
});
// → "0x9f3c…a7b1"
4
Confirm + return Pharoscan link
block 2,184,103 · status success · token #1
const receipt = await publicClient.waitForTransactionReceipt({ hash: txHash });

return {
  txHash,
  tokenId: "1",
  to: "0xRecipient",
  collection: "0xYourCollection",
  explorerUrl: `https://pharosscan.xyz/tx/${txHash}`,
};
